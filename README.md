# 漢字練習帳

`input.txt` に書かれた文字から重複するものを消して、漢字練習帳を自動生成します。

## 環境構築

```shell
npm ci
```

```shell
pip install pipenv
```

```shell
pipenv sync --dev
```

```shell
npm run setup
```

## 開発

```shell
npm run watch
```

このコマンドは、Python のスクリプトを自動で一定時間ごとに走らせます。

```shell
npm run preview
```

## ビルド

```shell
npm run build
```
